# UCII

## Универсальная инфраструктура криптографической идентичности

**Идентичность — это примитив. Аутентификация — это всего лишь приложение.**

Готовая к промышленному использованию инфраструктура криптографической идентичности, защищённая постквантовой криптографией и обеспеченная экономической авторизацией через x402.

---

## Обзор

UCII (Universal Cryptographic Identity Infrastructure — Универсальная инфраструктура криптографической идентичности) — это инфраструктура идентичности промышленного уровня, созданная для следующего поколения Интернета.

Вместо того чтобы рассматривать аутентификацию как основу системы, UCII рассматривает **идентичность** как постоянный криптографический примитив. Аутентификация, авторизация, учётные данные, платежи и доверие, специфичное для приложений, становятся сервисами, построенными поверх идентичности, а не определяющими её.

Каждая защищённая операция обеспечивается современной постквантовой криптографией и экономически контролируется через протокол оплаты x402.

UCII создана для обеспечения безопасной основы для людей, ИИ-агентов, автономных систем, организаций, сервисов и подключённых устройств.

---

## Почему UCII?

Традиционные системы идентичности строятся вокруг имён пользователей, паролей, OAuth-провайдеров и сессий аутентификации.

UCII использует другой подход.

Идентичность должна существовать независимо от способа её аутентификации.

Пароли устаревают.

OAuth-провайдеры меняются.

Сертификаты обновляются.

Методы аутентификации развиваются.

Идентичность должна оставаться постоянной.

UCII устанавливает криптографическую идентичность как корневой объект и позволяет методам аутентификации, учётным данным и механизмам авторизации развиваться без замены базовой идентичности.

Идентичность является постоянной.

Всё остальное строится поверх неё.

---

## Экономическая авторизация (x402)

UCII интегрирует протокол x402 для предоставления встроенной экономической авторизации инфраструктурных сервисов.

Каждая защищённая конечная точка публикует требуемый платёж.

Клиенты предоставляют доказательство оплаты.

UCII проверяет расчёт.

Квитанция расчёта сохраняется навсегда.

Попытки повторного использования отклоняются.

Только после этого выполняется защищённая операция.

Экономическая авторизация становится частью инфраструктуры, а не внешней системой выставления счетов.

## Публичный API

### Создание идентичности

```
POST /v1/identity
```

Создаёт новую универсальную криптографическую идентичность.

**Цена:** 0.50 USDC

---

### Регистрация учётных данных

```
POST /v1/credentials/register
```

Регистрирует криптографические учётные данные.

**Цена:** 0.75 USDC

---

### Проверка учётных данных

```
POST /v1/credentials/verify
```

Проверяет зарегистрированные учётные данные.

**Цена:** 0.10 USDC

---

Дополнительные конечные точки документированы через спецификацию OpenAPI.

## Соответствие постквантовой безопасности

UCII построена на современной постквантовой криптографии с использованием стандартизированных NIST алгоритмов, включая ML-DSA и ML-KEM.

Проект соответствует более широкому переходу индустрии к инфраструктуре, устойчивой к квантовым атакам, и предназначен для помощи организациям в принятии готовых к производству систем криптографической идентичности задолго до наступления сроков миграции на постквантовые технологии.

## Видение проекта

UCII создаёт инфраструктуру, а не приложение.

Идентичность должна быть универсальной.

Криптография должна быть защищённой от квантовых угроз.

Авторизация должна иметь возможность экономического контроля.

Разработчики должны иметь возможность создавать приложения поверх идентичности вместо того, чтобы заново создавать систему идентичности для каждого приложения.

UCII существует для предоставления этой основы.
