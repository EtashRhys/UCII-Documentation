# UCII Overview

## Universal Cryptographic Identity Infrastructure

UCII is cryptographic identity infrastructure designed for humans, AI agents, devices, services, robots, and organizations.

The core design principle:

> Identity is the primitive. Authentication is an application.

UCII separates permanent cryptographic identity from authentication methods, credentials, and authorization systems.

---

## Core Architecture

UCII provides:

- Universal cryptographic identity creation
- Post-quantum cryptographic credentials
- Credential lifecycle management
- Identity verification
- Machine identity support
- x402 economic authorization

---

## Identity Classes

UCII supports:

- HUMAN
- AI_AGENT
- ROBOT
- DEVICE
- SERVICE
- ORGANIZATION

---

## Economic Authorization

Protected infrastructure operations use x402.

The flow:

1. Client requests an operation.
2. UCII returns payment requirements.
3. Client submits payment authorization.
4. Settlement is verified.
5. Operation executes.
